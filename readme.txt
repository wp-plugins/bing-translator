A simple bing based translator plugin.



== Description ==

This plugin is a bing based translator.

You can display it two ways.
<ol>
<li>Widget</li>
<li>Shortcode (<code>[tb]</code>)</li>
</ol>

== Installation ==


1. Go to the Plugins> Add New> Upload in you admin dashboard.

2. Upload the zip file and install.
3. Go to the Plugins section in your admin dashboard and activate.
